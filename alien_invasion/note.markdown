# 创建游戏《外星人入侵》（alien_invasion)

# 规划项目
 - 描述项目，要实现的目标
 - 内容：《外星人入侵》中，
    - 玩家控制一艘在屏幕底部中央的飞船，
    - 使用方向键左右移动；
    - 空格键进行射击；
    - 开始后，一群外星人从屏幕中向下移动。
    - 玩家将屏幕上的外星人全部杀死后，出现新的外星人，而且速度更快
    - 外星人撞到飞船或到达屏幕底部，玩家丧失一条飞船
    - 玩家总共有3艘飞船，全部损失，游戏结束
    
# Pygame (Linux中，用pip方式安装)
 - pip是通用的Python包管理工具，包括对Python包的查找，下载，安装，卸载的功能
 - 安装Python中的pygame包：
    sudo apt-get install python3.6-dev mercurial
